* Uses `rspec`
* Uses [`database_cleaner`](https://github.com/DatabaseCleaner/database_cleaner) for ensuring a clean state for testing.
* Uses [`factory_bot`](https://github.com/thoughtbot/factory_bot) for replacing fixtures.
